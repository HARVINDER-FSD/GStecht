# Assets Folder

## Logo Setup

Please save your GS Tech IT Solutions logo as:
- **Filename**: `gs-tech-logo.png`
- **Location**: `project/public/assets/gs-tech-logo.png`
- **Recommended size**: 1200px width (high resolution for background)
- **Format**: PNG with transparent background (preferred) or white background

The logo will appear as a subtle watermark in the hero section background with very low opacity (4%).
